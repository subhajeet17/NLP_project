# NLP Preprocessing Application

This project demonstrates a complete NLP pipeline using Python and Streamlit.

## Features
- Tokenization
- Text Cleaning (Regex)
- Stemming
- Lemmatization
- Bag of Words
- TF-IDF
- Word Embeddings (spaCy)

## Tech Stack
- Python
- Streamlit
- NLTK
- spaCy
- Scikit-learn

## Run Project
```bash
pip install -r requirements.txt
streamlit run app.py
